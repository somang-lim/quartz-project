# Quartz 스케줄러 프로젝트

* Quartz 스케줄 CRUD 기능을 구현한 서비스입니다. 

## 개발 환경
* Eclipse 4.23 (2022-03)
* Java 8
* Maven 4.0
* Spring Boot 2.6.5

## 기술 스택
Spring Boot
* Spring Web
* Spring Quartz
* Spring Context Support
* Spring Boot Devtools

Database
* Tibero
* Oracle
* MyBatis 2.2.2

ETC
* Tomcat Embed Jasper
* Jstl
* Lombok
